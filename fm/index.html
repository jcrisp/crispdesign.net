Fractal Magic

Fractal Magic is a fractal drawing package. It draws the Julia and Mandelbrot sets and allows the user to zoom in on any point. It can print graphs and copy them to the clipboard for use in other applications. It is freeware - this means that you do not need to pay me to use it. Here is a screen-shot. Fractal Magic Requires:
- Microsoft Windows 95/98/NT
- 486 with 8meg of ram
- 256 colour capable monitor.
