 The demo version of Regarde is only 500K. You can't save or print graphs in the demo version of Regarde.

Download Regarde - Requires Windows 95 or later.

Unzip the file 'regdemo.zip' and run 'setup.exe'. If you do not have an Unzipper, you can download WinZip

If you want the old DOS version, DOS Regarde can still be downloaded. To use the DOS version, just unzip the two files (regarde.exe and splash.pcx) from regdos.zip and then run regarde.exe.

NOTE: Registration information provided by the Regarde demo may be not be up to date. Make sure you check out the online registration information.
