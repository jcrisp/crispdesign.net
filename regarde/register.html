Multiple User Licenses

If you would like to use Regarde on more than a single computer, you need to purchase a multiple user license.  Often educational institutions purchase licenses for all the computers in a computer lab so that a group of students can explore equation graphing as a class. Multiple user licenses are available at a discount.

The cost of multiple user licenses can be calculated with the formula:
cost = single user license cost * number of licenses * (1-discount)
