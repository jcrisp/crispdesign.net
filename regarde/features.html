 Graphing
  • Regarde is the only shareware program which can plot derivatives.
  • You can zoom in and out to see the whole graph at once or only a single part in more detail.
  • Horizontal and vertical zoom can be adjusted separately so that graphs can be stretched emphasize characteristics.
  • Numbering on the axes is completely customisable and can even be set to show distance in multiples of pi.
  • A grid can be turned on and off.
  • The coordinates of the mouse pointer are shown on the screen to provide an accurate way of reading values off graphs.
  • Equations do not need to be entered in any strange form. You just type them in as you would write them.


Interface

Regarde has an intuitive, colourful interface which caters to all people from novices to power users. Power users doRegarde can be used with the controlled completely with either the mouse or keyboard or a combination of both. This means that to enter the equation of a curve you can type it straight in from the keyboard or enter it with the onscreen buttons. Regarde has a flexible interface Help is also provided.



Printing and Saving

Regarde can print and save graphs. Graphs are saved in encapsulated postscript format (.eps) and can then be imported into word-processors (such as Microsoft Word), DTP packages (like Adobe Pagemaker) and maths note writing programs like Tek. Using Regarde WinPrint, a Windows utility that comes with the registered version, you can copy graphs to the Windows clipboard and print from windows. You can send curves straight to the printer from Regarde if you have a postscript printer.
