  • Can be dropped spilling contents.

Chemicals                                                 [ Modify ]

NaOH
  • Highly corrosive.
  • Can cause permanent blindness if got in eyes.
HCL
  • Highly corrosive.
  • Can cause permanent blindness if got in eyes.

Assessment of Chemical Risks           [ Modify ]

Risk: HCL and NaOH are highly corrosive and can cause blindness if splashed in eyes.  
Control Measures: Class will wear eye protection.
Risk: HCL and NaOH are highly corrosive and can skin damange.
Control Measures: Teacher will point out emergency shower and explain that this is to be used if HCL or NaOH comes into contact with skin.
