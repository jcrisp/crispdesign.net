Titration of NaOH with HCL

This is general experiment and cannot be used as a risk assessment.
You can Create a Personal Version of this Experiment if you want to use it for your class.


Text Book Reference
Science Spectrum, experiment 6.1.7.


Equipment

Conical Flask
  • Can be dropped spilling contents.
  • Breakage can lead to dangerous glass fragments which cut students.
Retort Stand
  • Can fall on student.

Chemicals

NaOH
  • Highly corrosive.
  • Can cause permanent blindness if got in eyes.
HCL
  • Highly corrosive.
  • Can cause permanent blindness if got in eyes.

Assessment of Chemical Risks

Risk: HCL and NaOH are highly corrosive and can cause blindness if splashed in eyes.  
Control Measures: Class will wear eye protection.
Risk: HCL and NaOH are highly corrosive and can skin damange.
Control Measures: Teacher will point out emergency shower and explain that this is to be used if HCL or NaOH comes into contact with skin.
